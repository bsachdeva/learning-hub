# Learning Hub

A portfolio-ready Next.js MERN course discovery app built with reusable React components.

## Setup
1. Copy .env.example to .env.local.
2. Run npm install.
3. Run npm run dev.
4. Open http://localhost:3000.

## Scripts
- npm run dev
- npm run build
- npm run start
- npm run test:unit
- npm run test:e2e
